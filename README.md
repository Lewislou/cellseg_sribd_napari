## Installation

You can install `cellseg-sribd` via [pip]:
```shell
conda create -y -n napari-env -c conda-forge python=3.9
conda activate napari-env
pip install "napari[all]"
cd cellseg_sribd_napari
pip install -r requirements.txt
pip install -e .
```
